# 🧾 Ηλεκτρονική Γραμματεία – Java Swing App με MySQL

📘 Μάθημα: Αρχές Τεχνολογίας Λογισμικού  

---

## 🔍 Περιγραφή

Η εφαρμογή "Ηλεκτρονική Γραμματεία" είναι ένα πληροφοριακό σύστημα γραφείου φοιτητών.  
Αναπτύχθηκε με Java (Swing) και MySQL. Οι χρήστες μπορούν να κάνουν login και να δουν πληροφορίες όπως:

- Προσωπικά δεδομένα
- Μαθήματα & καθηγητές
- Κατάσταση αιτήσεων (βεβαίωση, βαθμολογία κ.ά.)

Ο **διαχειριστής (admin)** έχει τη δυνατότητα να διαχειρίζεται τα δεδομένα της βάσης (προσθήκη, διαγραφή, ενημέρωση χρηστών).

---

## 🧩 Βασικά χαρακτηριστικά

- **Login** για φοιτητές & admin
- **Προβολή μαθημάτων και καθηγητών**
- **Διαχείριση χρηστών** (προσθήκη/ενημέρωση/διαγραφή) από admin
- **Διασύνδεση με βάση MySQL**
- **UML σχεδίαση** και **αντικειμενοστραφής προγραμματισμός (OOP)**
- **Γραφικό περιβάλλον (Swing)** με χρήση `JFrame`, `JPanel`, `JMenuBar` κ.ά.

---

## 🗂️ Δομή Πακέτων

- `classes.database`: Σύνδεση και συναρτήσεις προς τη βάση (queries, CRUD)
- `classes.gui`: GUI στοιχεία και κεντρικό menu
- `classes.person`: Οντότητες χρηστών (student, teacher, admin)
- `classes.services`: Βοηθητικές υπηρεσίες (π.χ. αιτήσεις)

---

## 🧪 Βάση Δεδομένων

- Υλοποιήθηκε σε **MySQL**
- Περιλαμβάνει πίνακες:
  - `users`, `teacher`, `student`, `lesson`, `teaches`, `enrollment`, `prerequisite`
- Δείτε το αρχείο `tei.sql` για δημιουργία και δοκιμαστικά δεδομένα

---

## ▶️ Εκτέλεση

1. Εισαγωγή της βάσης από το `tei.sql`
2. Άνοιγμα του project σε **NetBeans**
3. Εκτέλεση του main μέσω του αρχείου `menu.java`
4. Είσοδος με κάποια από τα υπάρχοντα credentials (δες το `sql` για χρήστες)

---

## 📎 Περιλαμβανόμενα αρχεία

- `src/` – Πηγαίος κώδικας Java
- `nbproject/` – Ρυθμίσεις NetBeans
- `tei.sql` – Βάση δεδομένων
- `Αναφορα.docx` – Αναφορά ομάδας και περιγραφή φάσεων
- `manifest.mf` – Manifest αρχείο για το .jar
- `build/` – (προτείνεται να διαγραφεί πριν το ανέβασμα)
